Olá!!!
Obrigado por ter baixado o BDJBALL...

Este é um jogo com código fonte aberto utilizado 
para estudar organização de código com classes.

Este código foi escrito inteiramente no 
DEV-C++ 4.9.9.2 com o auto indentar ligado e com o Tabulação Inteligente desligado.

ME desculpe caso a indentação esteja sendo exibida errada em seu compilador.
A indentação do DEV é muito ruim mesmo.


Você pode alterar o código fonte do jogo COMO QUISER.
Todo o código fonte e imagens utilizadas foram criadas por mim.

Como está é uma programação em C++, 
EU NÃO ME RESPONSABILIZO por danos causados ao seu computador, perda de memória ou travamento.

Eu não inclui o EXE, você mesmo deve compilar o código em seu DEV-C++.
Para isto, você precisa ter o Allegro 4.2.2 instalado.




Para ler o código, é preciso ter conhecimento em C++ e Orientação a Objetos.

ORDEM PARA LER O CÓDIGO E ENTENDER:

 - Primeiro, se você estiver usando o DEV, abra o bdjball.dev
 - Você vai começar lendo pelo arquivo MAIN.CPP

O main é o arquivo principal do jogo.
É ele que é executado por primeiro, antes de todos os outros fontes.

Leia o passo 1

No fim do passo 1 nós iniciamos a Allegro e algumas variáveis.
Você não precisa entender o que essas variáveis fazem ainda.


Leia o passo 2, no main iremos criar um objeto para cenario..
por isso, abra o arquivo cenario.h

Neste passo 2 aprendemos como criar uma classe e variáveis constantes.
Observe que criamos várias variáveis e alguns metodos que já tinhamos reconhecido
no começo do projeto.

Ao fim do passo 2, abra o arquivo cenario.cpp

Leia o passo 3

Você deve entender o que cada função da cenario faz até o fim de passo 3


Continue seguindo os passos para entender como o código fonte foi criado.
